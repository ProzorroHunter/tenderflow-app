# 🚀 ProzorroHunter — tenderflow-app

**Умный трекер тендеров Prozorro 24/7**  
Автоматический поиск по твоим параметрам + мгновенные уведомления в Telegram-канал.  
Стеклянный дизайн, удобный интерфейс, работает даже когда ты спишь 💤

## ✨ Возможности
- Поиск тендеров по ключевым словам, CPV, региону, бюджету, дате
- Автоматический мониторинг **каждые 10 минут** 24/7
- Уведомления в **твой Telegram-канал** (и личные сообщения)
- Shareable ссылки — подключай друзей/команду
- Красивый glassmorphism-дизайн (спокойные тона + светящиеся кнопки)
- Готово к монетизации (подписки в будущем)

## 🛠 Технологии (Вариант 2 — профессиональный)
- **Backend**: FastAPI + SQLAlchemy + Celery + Redis
- **Frontend**: React + Vite + Tailwind CSS (glassmorphism)
- **БД**: PostgreSQL
- **Telegram**: aiogram 3.x
- **Деплой**: Railway / Fly.io (бесплатно на старте)
